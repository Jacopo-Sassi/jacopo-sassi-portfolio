Progetti Selezionati

📌 Music Pulse – Music Streaming App

Obiettivo del progetto:
Sviluppare un player musicale cross-platform ad alte prestazioni che integra il catalogo reale di Spotify, focalizzandosi sulla gestione fluida dello stato globale e dell'audio in background.


Cosa dimostra:
Capacità di gestire flussi asincroni complessi, integrazione di API esterne con autenticazione OAuth2 (Client Credentials) e architettura mobile scalabile.


Scelte tecniche:

  • State Management: Uso di Zustand per un controllo granulare del player tra diverse schermate, evitando prop-drilling.
  • API Integration: Implementazione di Axios per comunicare con Spotify Web API, gestendo token di accesso e ricerca brani.
  • Custom Navigation: Architettura di navigazione a tab ottimizzata per mantenere il processo audio attivo durante il cambio schermata.
  • UX & Logic: Gestione di timeline interattive, animazioni fluide con Animated API e sistemi di fallback audio per licenze API limitate.
